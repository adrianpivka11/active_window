# Active Window Tracking

Professional Python module for project management and time tracking systems.

This repository contains a Python component used as part of a company internal application for **project management** and **time tracking**.

Its purpose is to provide a **secondary layer of time tracking** by monitoring active application windows while employees have running time tracking sessions.

Collected activity data is processed and stored in a **PostgreSQL database** for further reporting, analysis, and productivity insights.

---

## Overview

The module works in the background during active tracking sessions and records information about currently focused windows or applications.

This helps organizations better understand how tracked working time is spent across tools, tasks, and software environments.

---

## Key Features

- Monitor active windows during time tracking sessions
- Capture window/application usage data
- Process and structure activity records
- Send collected data to PostgreSQL database
- Support internal reporting and analytics
- Integrate with larger project management ecosystem
